# Video Speed Up Chrome Extension
## Beta POC
```
git clone https://github.com/jatr6994/Video-Speed-Up-Chrome
```

### In Google Chrome:
```
Navigate to chrome://extensions/
Enable Developer Mode
Click Load Unpacked
Load the unpacked service
```

### Speed Settings:
```
0.5x = half speed
1x = normal speed
10x = 10 times speed
```
